Motivatie

Ik heb voor deze opleiding gekozen, omdat ik op de middelbare school informatica leuk vond.
Hoe ga ik er voor zorgen dat ik gemotiveerd blijf?
Ik denk gewoon door alle opdrachten te doen, en door zelf ook wat dingen te schrijven die wat dieper gaan dan de opdrachten die we nu krijgen.
Wat ben ik bereid om na de lessen nog te doen?
Ik ga zelf ook wat dingen schrijven. In bijvoorbeeld javascript.

Mijn ontwikkelpunten

Ik wil het liefst zoveel mogelijk programmeer talen leren, zodat ik alles kan maken dat ik wil.
Mij lijkt het leuk om bugs te kunnen vinden in een code van iemand.
Ik wil ook leren om zelf dingen te onderzoeken, bijvoorbeeld hoe ik code efficienter kan schrijven.
